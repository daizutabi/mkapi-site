Parameters

Examples 
>>> from mkapi.core.base import Base
>>> doc = Docstring()
>>> parse_source(doc, Base)
>>> section = doc['Parameters']
>>> section['name'].to_tuple()
('name', 'str, optional', 'Name of self.')
>>> section = doc['Attributes']
>>> section['html'].to_tuple()
