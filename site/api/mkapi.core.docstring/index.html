MKAPI.CORE.DOCSTRING

This module provides functions that parse docstring.

FUNCTION
mkapi.core.docstring.section_header(line) → (str, str)

Returns a tuple of (section name, style name).

Parameters
  • line (str) — Docstring line.
Examples
>>> section_header("Args:")
('Args', 'google')
>>> section_header("Raises")
('Raises', 'numpy')
>>> section_header("other")
('', '')
GENERATOR
mkapi.core.docstring.split_section(doc) → Iterator[Tuple[str, str, str]]

Yields a tuple of (section name, contents, style).

Parameters
  • doc (str) — Docstring
Examples
>>> doc = "abc\n\nArgs:\n    x: X\n"
>>> it = split_section(doc)
>>> next(it)
('', 'abc', '')
>>> next(it)
('Parameters', 'x: X', 'google')
