FUNCTION
mkapi.core.node.is_member(name, obj, sourcefiles) → int

Returns an integer thats indicates if obj is a member or not.

  • -1 : Is not a member.
  • >0 : Is a member. If the value is larger than 0, obj is defined in different file and the value is corresponding to the index of unique source files of superclasses.
Parameters
  • name (str) — Object name.
  • obj (any) — Object
  • sourcefiles (list of str) — Parent source files. If the parent is a class, those of the superclasses should be included in the order of mro().
