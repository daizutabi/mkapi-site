Helper Function

Define a helper function that performs everything at one step.

def render(obj: Any):
    node = mkapi.get_node(obj)
    markdown = node.get_markdown()
    html = converter.convert(markdown)
    node.set_html(html)
    html = node.render()
    return HTML(html)
